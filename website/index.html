Everything you need

Simple, fast, and secure local file sharing

Auto Discovery

Devices find each other automatically using mDNS. No IP addresses, no configuration needed.

Any File Type

Send documents, images, videos, archives — anything. No file size limits on your local network.

Instant Messaging

Send quick text messages between devices. Perfect for sharing links, code snippets, and notes.

Private & Secure

Files never leave your local network. No cloud, no accounts, no data collection. Ever.

Cross Platform

Works on macOS, Windows, and Linux. Share files between any combination of devices.

Blazing Fast

Direct peer-to-peer transfers at full LAN speed. No internet bottleneck, no upload limits.

How it works

Three steps. That's it.

1

Install & Launch

Download Liminn and open it. It starts scanning your network immediately.

2

Select a Device

Other devices running Liminn appear automatically in the sidebar.

3

Send Anything

Type a message or attach a file. It arrives on the other device instantly.

Liminn is open source. Built with Electron, React, and TypeScript.
